Press Any Key to Start

How to Play

Simon will give the first signal. Repeat the signal by pressing the same color button. Simon will duplicate the first signal and add one. Repeat these two signals by pressing the same color buttons, in order. Continue playing as long as you can repeat each sequence of signals correctly. If you fail to repeat a sequence exactly, or if you take more than 5 seconds to repeat a signal, the game is over.
